脚本编写

基础

按键之间用单个空格分隔。
同一行中的内容会同时按下并释放。
要输入文本，请使用 STRING 函数。

示例 说明
WINDOWS
r 		先按下 Windows 键，再按 r 键
WINDOWS r 同时按下 Windows 键和 r 键
STRING WINDOWS r 输入文本 WINDOWS r

函数

命令 示例 说明
REM REM Hello World! 注释
DEFAULTDELAY 或 DEFAULT_DELAY DEFAULTDELAY 200 每条命令之间的默认间隔（毫秒）
DELAY DELAY 1000 延时（毫秒）
STRING STRING Hello World! 输入后面的字符串
REPEAT 或 REPLAY REPEAT 3 重复上一条命令 n 次
LOCALE LOCALE DE 设置键盘布局。可用值：DE, GB, US, ES, FR, DK, RU, BE, PT, IT, SK, CZ, SI, BG, CA-FR, CH-DE, CH-FR, HU
KEYCODE KEYCODE 0x02 0x04 输入指定的键码（修饰键, 按键1[, ..., 按键6]），可用十进制或十六进制
LED LED 40 20 10 修改 LED 颜色，RGB 十进制数值（0-255）
